# 🎯 Number Guessing Game (Java)

A simple console-based Number Guessing Game built using Java. The player has to guess a secret number. The program provides hints if the guess is too high or too low, counts the number of attempts, and allows the player to play multiple rounds.

---

## 📌 Features

- ✅ Console-based game
- ✅ User input using Scanner
- ✅ Unlimited guessing attempts
- ✅ "Too High" and "Too Low" hints
- ✅ Counts total attempts
- ✅ Play Again option
- ✅ Beginner-friendly Java project

---

## 🛠️ Technologies Used

- Java
- Scanner Class
- Loops (`while`)
- Conditional Statements (`if-else`)
- Variables

---

## 🚀 How to Run

1. Clone this repository:
   ```bash
   git clone https://github.com/bitanughara/Number-Guessing-Game.git
   ```

2. Open the project in VS Code or any Java IDE.

3. Compile the program:
   ```bash
   javac Number_Guessing_Game.java
   ```

4. Run the program:
   ```bash
   java Number_Guessing_Game
   ```
